# Threat Catalog Schema

Each synthetic threat record includes:
- `threat_id`
- `threat_name`
- `stride_category`
- `ai_threat_type`
- `affected_component`
- `affected_data_flow`
- `trust_boundary`
- `baseline_likelihood`
- `baseline_impact`
- `cloud_modifiers`
- `example_controls`
- `evidence_required`
- `synthetic`
